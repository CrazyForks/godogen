Use ${GODOGEN_COMMAND} to build or update this Babylon.js browser game from a natural language description.

This repo is a Babylon.js + TypeScript + Vite project. Work is interactive and the deliverable is a live URL: keep `npm run dev` running so the game stays at `http://127.0.0.1:5173` for the user to watch and steer, refreshing to see changes. Scenes are addressable as `http://127.0.0.1:5173/?scene=<name>`, with `main` as the default.
